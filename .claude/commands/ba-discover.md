description: Discover requirements — stakeholder mapping, elicitation, user research
---

Start discovery phase. Activate the following agents in sequence:

1. `@ba-identity` — Map stakeholders (RACI, Power/Interest Grid)
2. `@ba-strategy` — Analyze business context (SWOT, PESTLE, BMC)
3. `@ba-questioning` — Prepare interview questions (context-free + domain-specific)
4. `@ba-elicitation` — Execute deep-dive interviews (Funnel, SPIN, Colombo)
5. `@ba-ux` — User research (persona, journey map, empathy map)

Ask the user for:
- Project name and scope
- Known stakeholders (or "help me find them")
- Business context (or "research the domain")

Output: Stakeholder register, strategic context, interview questions, initial persona.

Handoff: `/ba-analyze` for deeper analysis, `/ba-specify` to write requirements.
